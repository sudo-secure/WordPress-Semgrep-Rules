📌 WordPress Semgrep Rules

⚡ Why this matters

WordPress powers a huge portion of the web, but many plugins and themes introduce security risks. These Semgrep rules are designed to automate security reviews and help developers catch issues early.



🚀 Usage
Install Semgrep:
pip install semgrep

Run scan:
semgrep --config path/to/rules


🎯 Who is this for?
Security researchers
WordPress plugin/theme developers
Bug bounty hunters
Code reviewers
